# Contributing

## Setup

1. `npm install`
2. `npm run dev`
3. Add your Anthropic key in the app to test conversation features.

## Development Rules

1. Keep PRs scoped and linked to one milestone/issue.
2. Update tests for behavior changes.
3. Run checks before opening PR:
   - `npm run lint`
   - `npm run build`
   - `npm run test`

## PR Checklist

- [ ] Architecture impact explained.
- [ ] Manual QA notes included.
- [ ] No secrets committed.
- [ ] Documentation updated.

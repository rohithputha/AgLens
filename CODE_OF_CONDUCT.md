# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free
experience for everyone.

## Standards

Examples of behavior that contributes to a positive environment:

- Respectful, constructive feedback.
- Clear technical reasoning in discussions.
- Collaboration with empathy and patience.

Examples of unacceptable behavior:

- Harassment or intimidation.
- Personal attacks.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove or edit comments, commits, issues, or other
contributions that violate this Code of Conduct.

## Reporting

Report issues privately to the maintainers.
